NN::NN ( const vector< unsigned > &  topology)

Constructs the neural network based on the given topology.

Neural network constructor that builds layers and initializes neurons.

Parameters
topologyA vector representing the number of neurons in each layer.

Each layer is constructed according to the topology provided. The constructor also creates bias neurons, which always output 1.0. Bias neurons help the network learn patterns that require a constant offset.
