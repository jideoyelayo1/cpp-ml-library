◆ getTopology()

void TrainingData::getTopology ( vector< unsigned > &  topology)

Reads the topology of the neural network from the training data.

Parses the topology of the neural network.

Parameters
topologyA vector to store the topology data.

This method reads a line from the training data file that starts with the keyword "topology:" followed by integers representing the number of neurons in each layer. For instance, a topology line of "topology: 3 2 1" describes a network with 3 neurons in the first layer, 2 in the second, and 1 in the final layer.
